# Сервис Продукта

## Запуск проекта
```
npm run dev
```

## Переменные окружения
```
HOST - на каком хосту запущен проект
PORT - порт проекта
PRODUCTS_COUNT - общее количество товаров
```
> [WARNING]
> PRODUCTS_COUNT -  временное переменное окружение используется для МОК
